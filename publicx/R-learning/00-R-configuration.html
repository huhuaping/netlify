1 设置包库的安装路径（packages library path）

R语言代码的分析和运行往往依赖于各类packages，首次使用时需要在线安装它们。强烈建议把这些packages安装在你指定的一个文件路径下（最好是非系统盘），例如D:\r material\Rpackages。

首先，你可以在Rstudio中输入如下代码，运行后可查看当前的包库目录。

# see your current lib path in Rastudio console 
.libPaths()
## [1] "D:/r material/Rpackages"           
## [2] "C:/Program Files/R/R-3.5.3/library"

如果你不满意这个包库的安装路径，你也可以设置成自己喜欢的目录（例如D:/software/Rpackages），此时则需要在Windows中系统选项下设置你的环境变量： （abmin权限下）控制面板-> 系统和安全 -> 系统 -> 系统属性 -> 高级 -> 环境变量 -> user的用户变量 -> R_LIBS_USER -> 编辑 -> （浏览目录） -> 确定。

Variable name: R_LIBS_USER 
Variable value: D:/software/Rpackages

1.1 设置工作文件路径（work directory path）。

我们的某项工作常常需要使用特定的工作文件夹，让Rstudio知道你指定的默认工作文件路径，可以让工作变得更轻松（例如，更有效地引用数据或图片文件等）。下面分情况而定：

  • 强烈建议总是以项目（project）的方式管理所有工作文件。这时，project所在的文件夹路径就自动成为你这个项目的默认工作文件路径。以后你只需要打开.Rproj文件即可。Rstudio中新建project的流程为：File -> New Project -> （设置路径...）：
